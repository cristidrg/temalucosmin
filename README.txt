Proiectul consta intr o platforma de genul "rent my car", in care lumea se autentifica si isi poate
lista masina pentru a fi inchiriata pentru o perioada determinata sau poate inchiria o masina din cele listate.
Momentan este creata doar pagina principala si pagina de Sign-up, urmand ca partea de autentificare sa fie realizata 
pe pagina principala folosind java script.
